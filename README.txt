This project contains two folders:

The nanopattern-tool folder contains a Swing UI for the nanopattern analysis tool, and the UploadServlet folder contains a Java Servlet front end.

The analysis tool written by Jeremy Singer is located in:
nanopattern_project/nanopattern_tool/src/uk/ac/glasgow/jsinger/nanopatterns/

The Swing UI files for this tool, written as part of the MSc in Information technology at Glasgow University, 2013, are located in: 
nanopattern_project/nanopattern_tool/uk/ac/glasgow/dlee/nanopatterns/
